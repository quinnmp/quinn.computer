SongScribe

This is a React-based website that makes use of the Spotify and Genius Lyrics APIs to create an environment in which users can take notes on songs as they listen to them. There are too many features to name here, see the README.md in the repository. It makes use of MongoDB to store user notes.

MILES Music

This is a blog I made to host my reviews of albums. I loaded this with features, such as timestamp buttons which make use of the YouTube API which allows users to listen to sections of songs embedded into the review. This also uses MongoDB to store reviews.

PicASCIIo

This is a silly little project I made which can convert uploaded images or drawn images into ASCII art. It makes use of mostly vanilla JS and MongoDB.
